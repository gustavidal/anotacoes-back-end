# Anotações da Disciplina de Back-End

## Atividades
- Atividade 01 (04/02/2026) -> Calculadora de juros compostos.
- Atividade 02 (13/02/2026) -> Calculadora de operações básicas (adição, subtração, multiplicação e divisão).
- Atividade 03 (25/02/2026 - 05/03/2026) -> Lista de exercícios envolvendo 5 atividades próprias envolvendo lógica (IMC, média escolar, tabuada, fatorial e par ou ímpar).
